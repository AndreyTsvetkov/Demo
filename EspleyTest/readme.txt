Тестовое задание звучит так: 
	Тестовое задание для программистов

	1. Написать приложение на любом вам знакомом языке программирования, которое будет собирать резюме с сайта jobs.uz (torg.uz или любого другого) по тематике "программирование и IT". Приложение должно собирать резюме и складывать их в базу данных (любую на ваше усмотрение).

	Пометки к задаче:

	1. Приложение должно быть написано с применением ООП

	2. Данное приложение должно быть WEB приложением. 

	3. Перед выполнением задачи дать эстимейт и придерживаться его.

	4. Вёрстка приложения будет являться плюсом.

Уточнение задачи: 

Также выяснилось, что подразумевается просмотр импортированных резюме в веб-интерфейсе. Так как никаких иных уточнений нет, и это тестовое задание, а не конечный продукт, ограничусь страничным выводом полученных резюме с возможностью просмотра.
Итого, получается на выходе нужны две страницы: список резюме и отдельное резюме. 

Кроме того, нужен компонент, который будет означенные резюме загружать с указанного сайта. 

Архитектура решения приведена в проекте Architecture.
